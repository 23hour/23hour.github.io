# 常用快捷键

Windows
多光标
Multiple selections (multi-cursor)		Alt + Click
块选
Column (box) selection		Shift + Alt + Dragging
Ctrl+Shift+Alt+Down / Up / Left / Right / PageDown / PageUp
选词Ctrl + D
选行Ctrl + L
切换头文件、源文件Alt + O
在下面插入一行
Insert line below		Ctrl+Enter
上下行对换Alt + Down / Up
复制一行Shift + Alt + Down
行注释Ctrl + /
块注释Shift +Alt +A
